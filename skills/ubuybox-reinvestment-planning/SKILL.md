# UBUYBOX Reinvestment Planning

## Overview

Use this skill after the first deals start producing cash. The current playbook recommends recycling a major chunk of wins back into the machine so the operation scales faster.

## Reinvestment Priorities

Allocate reinvestment into:

- marketing
- tools and systems
- training or coaching
- VA or call-support capacity when ready

The source notes suggest putting roughly 50% back into marketing when possible. Treat that as a planning benchmark, not an automatic rule.

## Workflow

1. Start with actual profits, not optimistic projections.
2. Protect the channels that produce more deal flow first.
3. Fund systems and training that improve conversion and consistency.
4. Add support capacity only when the workflow is ready for leverage.

## Output Format

Return:

- reinvestment categories
- suggested allocation logic
- stage-appropriate spending priorities
- risks of scaling too early or underinvesting

## Guardrails

- Do not overspend on support before the core funnel is proven.
- Do not present the 50% marketing number as mandatory.
- Tie reinvestment recommendations back to actual KPI and deal outcomes.
