# UBUYBOX KPI Review

## Overview

Use this skill to run the acquisitions process like a measurable business. The source plan explicitly treats KPI review as a habit, not a side task.

## Track These Metrics

- list size after filters
- dials per day
- contacts per day
- leads per day
- appointments
- contracts
- dispositions speed measured as days to assign or sell

## Review Rhythm

1. Review daily throughput metrics.
2. Review conversion bottlenecks.
3. Review monthly whether filters or target markets should change.
4. Feed the findings back into market selection and lead stacking.

## Output Format

Return:

- metric snapshot
- bottleneck diagnosis
- recommended operational change
- next review questions

## Guardrails

- Do not report vanity metrics without connecting them to outcomes.
- If the team is not measuring something critical, say so directly.
- Recommend filter or market changes only when the data supports them.
